# Clinic Check-In System

A digital registration platform designed to streamline the patient check-in process at medical clinics.

###  Key Features
* **Patient Registration:** Easy-to-use form for new and returning patients (`Register.vue`).
* **Instant Check-in:** Reduces waiting room congestion with a digital-first approach.
* **Modern UI:** Built with a clean, professional aesthetic suitable for healthcare environments.
